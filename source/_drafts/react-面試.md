## React
背景
- React 是 Facebook 在 2011 年开发的前端 JavaScript library。
- 使用 component 建構可重用的 UI
- 2015 年开源，但有一个很大的支持社区。

特色
- 使用虛擬 DOM
- 單向數據流、數據綁定
- 可使用server 渲染???

優點
- 提高效能
- 方便 client、server 端使用
- JSX 可讀性好
- 編寫UI TEST容易

限制
- React 只是一個 library，不是一个完整的框架
- 新手需要花時間理解，library 非常庞大，需要时间来理解
