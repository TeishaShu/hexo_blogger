### template literals 模板字串
使用重音符「`」在字的前後，變數使用「${變數}」
就不用使用「+、''、""」
```
const list = document.querySelector('.list');
const imgUrl = 'logo.png';
const title = '我是標題';
list.innerHTML = `<li>
  <h2>${title}</h2>
  <img src="${imgUrl}" alt="">
<li>`
```
